# Restaurant Control Portal

Multi-user restaurant management web portal with stock control, expenses, cash management and role-based permissions.

## Features
- Multi-user login with role permissions
- Custom stock sections with minimum stock alerts
- Manual expense entries
- Cash management
- Dashboard with low stock alerts
- Data persistence in localStorage
